# Usage
1. Fork the repo and then clone it or download it.

2. First install all dependencies:
    ```bash
    # with npm
    npm install
    
    # or with yarn
    yarn
    ```
3. Start the server
    ```javascript
    npm run start  
    ```
    dev server
    ```javascript
    npm run start:dev  
    ```

4. Now run the app
    ```javacript
    localhost:[PORT] 
    ```
